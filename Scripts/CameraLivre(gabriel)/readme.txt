Os scripts FPSCamera.cs e DragController.cs são colocados na câmera.

O script OutlineManager.cs é colocado no objeto que vai ser mexido ex: PC, caixa dos órgãos

Se a outline aparecer bugada carreguem nos prefabs dos models na pasta e no import settings ativem o Read/Write.

Se o PC estiver a dar problema vão aos Project Settings -> Player -> Other Settings e desativem o Static Batching.
